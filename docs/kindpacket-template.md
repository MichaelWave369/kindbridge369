# KindPacket Template

Use this template to create a new printable KindPacket.

## Packet title

Example: My power might get shut off

## This is for

Describe the moment in one calm sentence.

## First 5-minute step

Give one action someone can take right now.

## Gather if you can

- Item 1
- Item 2
- Item 3
- Item 4

## Words to use

> Hi, I am trying to handle this responsibly. Can you explain what options are available and what I should do next?

## Do not panic about

Name one thing that feels scary but can be handled step by step.

## Search terms

- term near me
- county/state program name
- local nonprofit category

## Professional or emergency gate

Explain when this needs qualified help, emergency support, legal aid, medical care, crisis support, or an official agency.

## Local notes

Add city/county/state-specific notes only if they can be verified and maintained.
