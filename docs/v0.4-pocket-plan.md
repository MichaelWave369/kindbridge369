# v0.4 Pocket Plan

Pocket Plan is the next KindBridge369 feature direction.

Goal: let a person choose one situation, add one contact target, write one tiny promise, and print or copy a one-page next-step plan.

## Fields

- Situation
- First step
- Things to gather
- Plain words to use
- Suggested KindPacket
- Person or place to contact
- Tiny promise for today
- What was said / receipt

## Design rules

- No account required.
- No server storage.
- No tracking.
- Printable first.
- Copyable text second.
- Calm language only.
- The user stays in charge.

## Future wiring

Add this as a new module tab between HelpFinder and KindPackets once the app shell is ready for a small refactor.
