# KindBridge369 v0.5 — Print Shelf

The v0.5 pass adds a static printable shelf under `public/print/`.

These pages are copied directly into the deployed Vite site, which means they do not depend on React routing, account systems, databases, or third-party services.

## Public paths after GitHub Pages deploy

Depending on the repository Pages URL, these should be available at paths like:

- `/kindbridge369/print/`
- `/kindbridge369/print/pocket-plan.html`
- `/kindbridge369/print/town-starter-kit.html`
- `/kindbridge369/print/resource-call-sheet.html`
- `/kindbridge369/print/neighbor-help-card.html`

## Added tools

### Pocket Plan

A printable one-page worksheet for turning a hard moment into one small next-step plan.

Fields include:

- What is happening
- Who or what place can help
- First step
- Gather if possible
- Plain words to use
- Tiny promise for today
- Receipt line

### Town Starter Kit

A printable worksheet for building a local help map with verified entries.

It focuses on:

- Need buckets
- Place or person
- Contact route
- What to bring
- Date checked

### Placeholder pages

Two simple placeholder pages were added so the Print Shelf has no broken links:

- Resource Call Sheet
- Neighbor Help Card

Those can be expanded in a later pass.

## Why this matters

Static printable pages are low-risk, high-usefulness assets. A person can open them, print them, fill them in by hand, copy them for a meeting table, translate them, or adapt them for a town without needing a backend.

This keeps KindBridge369 useful even before every React module is fully wired.
