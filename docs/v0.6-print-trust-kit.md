# KindBridge369 v0.6 — Print Trust Kit

v0.6 expands the `public/print/` shelf with trust-first pages that can be used without touching the React app, adding accounts, collecting data, or depending on a backend.

## Added pages

- `quick-start.html` — a one-page flow for naming a need, choosing a bucket, gathering useful details, and picking one small next step.
- `privacy-boundaries.html` — a dignity and consent sheet for helpers, including minimum-note habits and verification boundaries.
- `verification-log.html` — a table for tracking what local resource details were checked, when they were checked, and what remains unknown.

## Why this matters

KindBridge should be useful in places where people may only have a printer, a phone, a library computer, or a trusted helper sitting nearby. The print shelf makes the project useful even when the full app is not the best tool for the moment.

## v0.6 principle

Checked beats guessed.

Helpers should verify local details, date their checks, leave unknowns marked as unknown, and avoid writing private information that is not needed for the next step.

## Build impact

These files live in `public/print/`. Vite copies them to the final build as static assets. This keeps the React app stable while expanding the public-use toolkit.
